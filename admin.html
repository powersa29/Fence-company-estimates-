Deployment Steps

  1. Add your Google Maps API key to js/config.js (DEFAULTS section)
  2. Upload this entire folder to your web hosting or GitHub Pages
  3. Enter your hosted URL in Step 1 above
  4. Fill in each client's details and copy their embed code
  5. Paste the code anywhere in the client's website HTML

Hosting Options (all free)

  1. GitHub Pages — Push to a GitHub repo, enable Pages in Settings → instant free hosting
  2. Netlify — Drag-and-drop the project folder at app.netlify.com
  3. Any web host — Upload via FTP, no server-side code needed (pure static files)

API Key Restriction (important)

  1. In Google Cloud Console, find your API key under Credentials
  2. Set Application restrictions to "HTTP referrers"
  3. Add https://yourdomain.com/*
  4. This prevents anyone else from using your key, even if they see it
