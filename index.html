Google Maps API Key Required

The estimator needs a valid Google Maps API key to display the map.

  1. Go to console.cloud.google.com
  2. Enable Maps JavaScript API and Places API
  3. Create an API key under Credentials
  4. Open js/config.js and set googleMapsApiKey in the DEFAULTS section
  5. Or use the Admin panel to pass the key via URL parameter
